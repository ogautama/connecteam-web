Design reference

CONNECTeam design tokens & mobile mockups

The palette, type scale, and layout tokens shipped in src/app/globals.css, plus a mobile-first look at how the shipped layouts read on a phone — including a proposed nav pattern for the gap the current build has on small screens (the header nav and member sidebar both disappear below the md breakpoint with no mobile replacement yet).

See also: alternative mobile nav — bottom tab bar for the member area · rejected homepage direction — kept as a possible /learn-more seed

Color

Navy anchors the credible-insurance side of the brand; red is the interactive accent; yellow is a sparing highlight. Each scale is generated from one exact brand hex.

brand-navy (base: 700 = #183F87)
brand-red (base: 500 = #F04975)
brand-yellow (base: 400 = #F5BA01)
