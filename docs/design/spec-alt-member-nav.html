Design reference — alternative

Alt: member nav as a bottom tab bar

An alternative to the slide-in drawer in the primary spec: a bottom tab bar with the four most-used sections always visible, and a "More" tab that opens a sheet with the rest. Neither pattern is shipped code — this is a comparison to pick from before building it.

← Back to primary spec (tokens + drawer nav)

Drawer (primary spec)

All 9 sections in one list behind ☰. Familiar, scales to any nav length, but the full list is one extra tap away every time.

Bottom tab bar (this page)

Top 4 sections one thumb-reach away at all times; everything else lives behind "More." Faster for daily use, but needs a judgment call on which 4 sections "win."
