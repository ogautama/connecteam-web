Quest hub mockup — /member/onboarding

Static mockup for Plan 07's redesign: a brand-colored recreation of a UX reference video the team was shown (a gamified "quest" prototype — purple/pink gradient, level tabs, progress bars, accordion checklists). Colors here use the site's existing brand tokens (navy #183f87, red #f04975, gold #f5ba01, see src/app/globals.css) instead of the reference's generic purple/pink. Not shipped code — a reference to build from, same convention as spec-alt-member-nav.html.

← Back to primary spec (tokens + drawer nav)

💡 What's clickable in this mockup: the 5 tabs switch panels, and every accordion row expands/collapses and toggles its checkbox — it's a real interaction demo, not a screenshot. Only the Onboarding tab has real copy (migrated from src/content/onboarding.ts); the other 4 show the "Segera hadir" placeholder pattern this plan ships with.
